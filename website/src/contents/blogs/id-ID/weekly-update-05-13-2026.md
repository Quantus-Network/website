Penambang Bitcoin awal dapat menambang ribuan BTC sehari di laptop.

Penambangan CPU Bitcoin HANYA layak selama kurang lebih 2 tahun.

Seiring pertumbuhan jaringan, begitu pula hashrate. Penambangan GPU menjadi lebih layak, kemudian ASIC.

Pada saat Anda pertama kali mendengar tentang BTC, penambangan pada perangkat keras konsumen sudah mati.

Tidak ada bursa untuk membeli.

Karena tidak ada pasar sekunder, tidak ada "harga".

Untuk mendapatkan "harga", Anda memperkirakan biaya listrik per koin.

Pada saat Anda bisa membeli di pasar, sebagian besar koin sudah didistribusikan.

Bitcoin memancarkan lebih dari separuh koinnya dalam empat tahun pertama.

Semuanya diberikan kepada penambang awal.

Jadi itulah standar untuk "peluncuran yang adil".

Kita mungkin bisa melakukan yang lebih baik.

Saat kami bersiap untuk meluncurkan [@QuantusNetwork](https://x.com/@QuantusNetwork), kami memprioritaskan pembuatan penambangan yang dapat diakses sejak hari pertama.

Anda dapat menjalankan node penambangan di testnet kami hari ini dalam beberapa klik.

Minggu lalu kami memperbarui aplikasi penambang GUI kami. Bahkan jika Anda tidak teknis, Anda dapat bersiap untuk penambangan mainnet pada hari pertama.

### Core Tech & ZK:

- Membuat prototipe untuk pembuktian ZK yang didelegasikan di penambang. Fitur eksperimental ini akan memungkinkan penambang untuk memilih agregasi bukti untuk transaksi wormhole sebagai peran tambahan.
- Menangani temuan audit HashCloak dan Eiger.

### Network & Infra:

- Mengirimkan pembaruan penambang GUI dengan fitur penebusan dan saldo untuk testnet Planck. Unduhan dan panduan di docs (dot) quantus (dot) com
- Menerapkan kerangka kerja audit keamanan berbasis AI di semua pull request. Menyelesaikan tinjauan keamanan rantai penuh otomatis pertama.
- Melakukan pen-test pada node RPC.

### Web & Mobile App Updates:

- Mengimplementasikan layar hadiah penambang dan layar point-of-sale di dompet.
- Mengoptimalkan kueri explorer yang mahal dan memperbarui acara akun terpadu.
- Perbaikan aplikasi seluler.
