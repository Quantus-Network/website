Setiap chain mengklaim akan menjadi post-kuantum.

Jadi kami membangun papan skor.

[Migration.fail](https://migration.fail/) melacak persentase kapitalisasi pasar kripto yang berjalan pada kriptografi tahan kuantum.

Saat ini 0,046%.

Hampir nol.

Lebih dari $2T harus pindah dari alamat pre-kuantum dan menjadi post-kuantum.

Jadi kami membangun tujuannya.

Kami merilis v1.5.5 aplikasi Quantus dengan dukungan dompet perangkat keras Keystone.

Sekarang QUAN memiliki dukungan multisig dan cold storage.

Semuanya open source di GitHub.

Seluruh industri perlu beralih ke post-kuantum — sekarang Anda bisa memantau progres secara real time di [migration.fail](https://migration.fail/).

Berikut yang lainnya yang kami bangun:

### Core Tech & ZK

- Menggabungkan agregasi bukti lapisan kedua di sirkuit ZK, memungkinkan kami mengagregasi bukti yang sudah diagregasi untuk peningkatan throughput berikutnya.
- Memotong puncak memori untuk penandatanganan ML-DSA-87 dengan streaming matriks kunci, sehingga Dilithium praktis di perangkat embedded seperti Keystone.
- Mengeras resharing tanda tangan threshold dan meningkatkan perilaku waktu konstan di pustaka Dilithium kami.
- Menyelesaikan temuan medium dan low dari tinjauan internal chain, mengeras primitif FRAME dan dependensi.
- Memperbaiki temuan tinjuan internal tambahan di kode hashing Poseidon.
- Memperbaiki aset tertahan yatim pada pemulihan reversible-transfer di chain.
- Menambahkan suite smoke dan fuzz test quantus exercise ke Quantus CLI, plus fitur update.

### Network & Infra

- Merilis miner v3.3.0 dan v3.3.1 dengan dukungan GPU Windows yang lebih baik dan perbaikan pemilihan adapter GPU untuk duplikasi multi-backend dan error kehabisan memori.
- Meningkatkan alur setup mining dan menambahkan skrip setup mining ke situs docs.
- Menambahkan dukungan event tech governance ke Subsquid

### Web & Mobile App Updates

- Merilis v1.5.5 aplikasi seluler ke App Store dengan dukungan dompet perangkat keras Keystone: update firmware, dukungan multisig di firmware, pengerasan keamanan penanganan kunci, dan perbaikan bug integrasi. Fork firmware kami open source di GitHub.
- Menyelesaikan semua temuan medium dari tinjauan internal firmware Keystone, menambahkan tes conformance chain, hedged signing, dan device binding.
- Meluncurkan [migration.fail](https://migration.fail/).
- Menambahkan dukungan beberapa seed phrase ke dompet seluler.
- Merefaktor alur kirim dan merilis perbaikan UX di seluruh aplikasi seluler.
- Mengeras alur salin seed phrase dan parsing transaksi, serta menambahkan feedback haptik pada input jumlah.
- Merilis perbaikan kebenaran dari tinjauan internal runtime v12.
