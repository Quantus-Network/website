Каждый блокчейн говорит, что станет постквантовым.

Поэтому мы построили табло.

[Migration.fail](https://migration.fail/) отслеживает долю крипторынка, работающую на квантостойкой криптографии.

Сейчас это 0,046%.

По сути ноль.

Более $2 трлн нужно перевести с преквантовых адресов и сделать постквантовыми.

Поэтому мы строим пункт назначения.

Мы выпустили v1.5.5 приложения Quantus с поддержкой аппаратного кошелька Keystone.

Теперь QUAN поддерживает и мультиподпись, и холодное хранение.

Всё с открытым исходным кодом на GitHub.

Всей индустрии предстоит перейти на постквантовую криптографию — прогресс в реальном времени можно смотреть на [migration.fail](https://migration.fail/).

Вот что ещё мы построили:

### Github Activity (22 merged pull requests):

![Github Stats](/blog/assets/github-stats-weekly-update-07-09-2026.webp)

### Core Tech & ZK

- Объединили агрегацию доказательств второго уровня в ZK-схемах, что позволяет агрегировать уже агрегированные доказательства для дальнейшего роста пропускной способности.
- Снизили пиковое потребление памяти при подписи ML-DSA-87 за счёт потоковой передачи ключевой матрицы, что делает Dilithium практичным на встроенном оборудовании вроде Keystone.
- Усилили перераспределение пороговых подписей и улучшили поведение с постоянным временем в нашей библиотеке Dilithium.
- Устранили находки средней и низкой степени риска из внутреннего обзора цепи, усилив примитивы FRAME и зависимости.
- Исправили дополнительные находки внутреннего обзора в коде хеширования Poseidon.
- Исправили «осиротевшие» удерживаемые активы при восстановлении обратимых переводов в цепи.
- Добавили smoke- и fuzz-тесты quantus exercise в Quantus CLI, а также функцию обновления.

### Network & Infra

- Выпустили miner v3.3.0 и v3.3.1 с улучшенной поддержкой GPU в Windows и исправлениями выбора GPU-адаптера для дублирования multi-backend и ошибок нехватки памяти.
- Улучшили процесс настройки майнинга и добавили скрипт настройки майнинга на сайт документации.
- Добавили поддержку событий tech governance в Subsquid

### Web & Mobile App Updates

- Выпустили v1.5.5 мобильного приложения в App Store с поддержкой аппаратного кошелька Keystone: обновления прошивки, поддержка мультиподписи в прошивке, усиление безопасности обработки ключей и исправление ошибки интеграции. Наш форк прошивки с открытым исходным кодом на GitHub.
- Устранили все находки средней степени риска из внутреннего обзора прошивки Keystone, добавив тесты соответствия цепи, hedged signing и привязку устройства.
- Запустили [migration.fail](https://migration.fail/).
- Добавили поддержку нескольких seed-фраз в мобильный кошелёк.
- Рефакторили поток отправки и выпустили UX-исправления по всему мобильному приложению.
- Усилили поток копирования seed-фразы и разбор транзакций, добавили тактильную обратную связь при вводе суммы.
- Выпустили исправления корректности из внутреннего обзора runtime v12.
