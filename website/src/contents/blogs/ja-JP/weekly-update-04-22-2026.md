プランクは、Quantus のポスト量子プライベート取引に対応した最初のテストネットだ。

今週、マイナーの半数以上がプランクへ移行した。

他チェーンが量子脅威の真偽を議論している間に、我々は公開の場でソリューションをストレステストしている。

マイニング報酬は、CLI からモバイルウォレットの透明アドレスへ請求できるようになった。

Apple Silicon と Linux でマイニングをエンドツーエンドで検証済みだ。

現時点ではモバイル UI は透明アドレス型のみサポート。続報はまもなく。

[docs.quantus.com](https://docs.quantus.com) が公開され、ノード構築・マイナー実行・モバイルウォレットへの報酬請求までを案内するマイニングガイドを掲載している。

過去の Quantus テストネットでマイニングや利用をしていた場合、アップデート時にアプリから移行を促される。

残高はリセットされるが、設定の「mining rewards」を見れば、過去のテストネットで採掘した各ブロックのクレジットは残っている。

プランクではアカウントごとの採掘ブロック数も追跡している。

集約プローバの速度も、今週 2x8 分割アーキテクチャで倍になった。

その他にも今週は多く出荷した。

### Core Tech & ZK

- 2x8 分割証明アーキテクチャで集約プローバ性能を倍増（各 8 リーフバンドル単体では ZK なし、ルート証明に ZK ラッパー）。
- ZK 研究論文をプレプリント投稿。
- qp-plonky2 の新 split mask ZK オプションにプライバシー漏えいバグを発見しパッチ。
- 閾値 Dilithium の監査項目に対応。
- マイニング報酬を CLI で請求可能に。
- CLI のハッシャーを blake2-256 に更新。
- CLI のワームホール残高リカバリ修正をリリース。
- Tech Collective メンバーがランタイム更新を実行成功。
