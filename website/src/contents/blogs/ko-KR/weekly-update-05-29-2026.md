이번 주 추가로 출시한 내용:

### 코어 기술 및 ZK

- qp-plonky2, qp-poseidon, qp-rusty-crystals 전반의 감사 이슈를 해결했습니다.
- Poseidon2 게이트를 최적화하고, ZK 회로 와이어 구성을 강화했으며, qp-zk-circuits에 메모리 프로파일링 도구를 추가했습니다.

### 네트워크 및 인프라

- 마이너가 AI 추론 워크로드와 GPU를 공유할 수 있도록 선택적 GPU 마이너 스로틀링을 추가했습니다.
- 체인 난이도 조정 동작을 수정하고 체인 노드 Docker 이미지 지원을 개선했습니다.
- Subsquid에 멀티시그 이벤트 데이터, 총 마이너 수, API 속도 제한 상향을 추가했습니다.

### 웹 및 모바일 앱 업데이트

- 모바일 지갑 언어 지원, 당겨서 새로고침, 무한 스크롤 수정을 추가했습니다.
- 모바일 POS 모드 및 pay/send 인텐트 처리를 수정했습니다(콜드 스타트 처리, 소수 구분자 검증 포함).
- q (dot) day에서 스폰서 문의용 이메일 연동이 포함된 새 Q-Day 사이트를 배포했습니다.
- 이벤트 및 NS에서 QUAN 테스트넷 토큰으로 굿즈를 판매하는 Quantus 샵을 배포했습니다.
- Quantus 익스플로러 마이닝 리더보드 수정.
