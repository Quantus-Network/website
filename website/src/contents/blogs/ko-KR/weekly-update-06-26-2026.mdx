정장 차림도 Q-pilled 되고 있다.

이번 주 BlackRock은 양자 컴퓨팅과 블록체인에 관한 보고서를 발표했고, 백악관은 PQC 마이그레이션 일정을 앞당기는 행정명령을 내렸다.

둘 다 다음을 확인한다:

- 포스트 양자로 가는 것은 실행 문제다(더 이상 이념 논쟁이 아니다).
- 타원곡선 암호는 위험에 처해 있다.
- NIST 표준(ML-KEM, ML-DSA, SLH-DSA)이 해답이다.
- 어려운 부분은 조율이다.
- 마이그레이션은 5년 이내에 완료되어야 한다.

암호 시장에서 2조 달러 이상이 위험에 있으며, Q-Day 이전에 가능한 한 많이 보호하는 것이 우리의 목표다.

### Core Tech & ZK:

- ZK 회로와 Plonky2 일부에 대한 형식 검증을 진행했다.
- 간헐적 임시 reorg를 유발하던 합의 버그를 수정했다.
- 보안 감사에 앞서 runtime의 보안 핵심 코드를 단일 감사 가능 표면으로 통합했다.
- Web3 통합을 쉽게 하기 위한 JavaScript SDK를 출시했다.
- runtime 보안을 강화하고 runtime 코드 표면을 검토했다.

### Network & Infra:

- 마이너 알림: 최근 runtime 업그레이드로 보상을 청구하려면 CLI 업데이트가 필요하다.
- 노드 업데이트를 위한 Subsquid 배포 전략을 개선했다.

### Web & Mobile App Updates:
