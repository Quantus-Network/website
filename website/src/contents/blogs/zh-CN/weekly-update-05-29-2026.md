本周其他交付内容：

### Core Tech & ZK（核心技术与零知识证明）

- 处理 qp-plonky2、qp-poseidon 和 qp-rusty-crystals 中的审计问题。
- 优化 Poseidon2 门电路，收紧 ZK 电路连线配置，并在 qp-zk-circuits 中新增内存分析工具。

### Network & Infra（网络与基础设施）

- 新增可选 GPU 矿工节流，便于矿工与 AI 推理工作负载共享 GPU。
- 修复链难度调整行为，改进链节点 Docker 镜像支持。
- 在 Subsquid 中新增多签事件数据、矿工总数，并提高 API 速率限制。

### Web & Mobile App Updates（网页与移动端更新）

- 移动钱包新增语言支持、下拉刷新与无限滚动修复。
- 修复移动 POS 模式及 pay/send intent 处理，包括冷启动处理与小数分隔符校验。
- 在 q (dot) day 部署新版 Q-Day 站点，集成赞助商咨询邮件。
- 部署 Quantus 商店，在活动和 NS 用 QUAN 测试网代币销售周边商品。
- Quantus 浏览器挖矿排行榜修复。
