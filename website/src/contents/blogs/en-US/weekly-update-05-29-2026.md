Here's what else we shipped this week:

### Core Tech & ZK

- Addressed audit issues across qp-plonky2, qp-poseidon and qp-rusty-crystals.
- Optimized the Poseidon2 gate, tightened ZK circuit wire configuration and added memory profiling tooling to qp-zk-circuits.

### Network & Infra

- Added optional GPU miner throttling so miners can share a GPU with AI inference workloads.
- Fixed chain difficulty adjustment behavior and improved chain node Docker image support.
- Added multisig event data, total miner count and increased API rate limits in Subsquid.

### Web & Mobile App Updates

- Added mobile wallet language support, pull-to-refresh and infinite scroll fixes.
- Fixed mobile POS mode and pay/send intent handling, including cold-start processing and decimal separator validation.
- Deployed the new Q-Day site with email integration for sponsor inquiries at q (dot) day.
- Deployed the Quantus shop for selling merch for QUAN testnet tokens at events and NS.
- Quantus explorer mining leaderboard fixes.
