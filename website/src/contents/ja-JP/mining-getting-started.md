# Quantus でのマイニング

Quantus は、標準的なコンピュータを持つ人なら誰でもネットワークの保護に参加できるハイブリッド・コンセンサスメカニズムを採用しています。

## 要件

- **CPU/GPU**: ほとんどの最新ハードウェアで可能です。
- **Quantus ノード**: GitHub から最新のソフトウェアをダウンロードしてください。
- **安定性**: 安定したインターネット接続。

## なぜマイニングするのか？

マイニングすることで、ネットワークの分散化を助け、攻撃から保護することができます。見返りとして、QUAN トークンの形でブロック報酬を得ることができます。
