権力と権限を複数の当事者間で分割。分散型コーディネーション。

しかし、耐量子署名はビットコインの署名よりも 20 倍から 80 倍大きくなります。従来のマルチシグソリューションにも役割はありますが、特定のユースケースでは許容できないオーバーヘッドが発生します。

今週、私たちはしきい値 Dilithium 署名をリリースしました。これにより、分散されたグループがトランザクションに署名し、オンチェーンで単一の署名を生成することが可能になります。基本的には、秘密鍵のシャーディングです。

これは、カストディアン、クロスネットワークアプリケーション、およびチェーンの安全な自動利用のために不可欠です。

4 件のマージされたプルリクエストを含む、週刊 Github アクティビティレポート: https://github.com/Quantus-Network/n8n-workflows/blob/main/github/weekly-update-2026-01-13-10:13:26.md

## コア技術と ZK

- 安全な分散署名を可能にするしきい値署名を実装しました。
- ZK 回路のオーバーヘッドを削減しました。
- wormhole サンプルバイナリに証明アグリゲーションのサポートを追加し、チェーンおよび CLI への統合を進めています。
- ZK 回路と Poseidon ハッシュアルゴリズムをアップグレードし、すべてのデータ型を汎用的に処理できるように（汎用ストレージハッシュ）なり、チェーンおよび CLI に統合されました。
- ZK 回路内でのアセット ID に対する証明アグリゲーションのサポートを追加しました。
- トランザクション処理の高速化のために CLI を最適化し、高セキュリティのリカバリスクリプトを追加しました。

## Web およびモバイルアプリのアップデート

- CPU と GPU の使用量を細かく設定できるスライダーを備え、ハッシュレートレポートの修正を行った新しいバージョンの UI マイニングアプリをリリースしました。
- モバイルアプリへの完全なハードウェアウォレットサポートを確定し、マージしました。
- 単一のモバイルウォレットインスタンス内で複数のアカウントを管理できるサポートを追加しました。
- 高セキュリティ画面を実装し、「ガーディアン (Guardian)」リカバリフローの開発を開始しました。
- 「送信」画面の UX を洗練させ、エラー処理を改善し、認証タイマーのバグを修正し、アプリのアイコンを統一しました。
- 毎日のデータベースバックアップを設定し、システムサービスを修正し、Task Master コード（ウォレットアプリのイベント追跡）をクリーンアップしました。
