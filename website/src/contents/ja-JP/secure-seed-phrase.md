# シードフレーズを安全に保つ方法

シードフレーズは、あなたのすべてのデジタル資産への鍵です。分散型の世界には、「パスワードを忘れた」ボタンはありません。

## ベストプラクティス

1. **デジタル化しない**: シードフレーズをコンピュータ、携帯電話、クラウドストレージに絶対に保存しないでください。
2. **物理的なバックアップ**: 紙に書くか、金属に刻んでください。
3. **分割して保管**: フレーズの各部分を、別々の安全な場所に保管してください。
4. **ガーディアン**: Quantus の「ガーディアン」機能を使用して、ソーシャルリカバリの層を追加してください。

覚えておいてください：あなたの鍵でなければ、あなたのコインではありません。
