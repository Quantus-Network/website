# Penjelasan ZK-Proofs

Zero-Knowledge Proofs (ZK-Proofs) adalah terobosan dalam kriptografi yang memungkinkan satu pihak membuktikan kepada pihak lain bahwa suatu pernyataan adalah benar tanpa mengungkapkan informasi apa pun selain validitas pernyataan itu sendiri.

## Bagaimana Quantus Menggunakan ZK-Proofs

Di Jaringan Quantus, kami menggunakan ZK-SNARK rekursif untuk mengompres seluruh riwayat blockchain menjadi bukti kecil.

### Manfaat:
- **Skalabilitas Masif**: Node tidak perlu menjalankan ulang setiap transaksi.
- **Privasi**: Transaksi dapat diverifikasi tanpa mengekspos data sensitif.
- **Node Ringan**: Jalankan node di smartphone Anda!

Ikuti terus pendalaman teknis kami tentang arsitektur SNARK rekursif kami.
