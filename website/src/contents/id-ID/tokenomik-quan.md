# Tokenomik QUAN

Token QUAN adalah darah dari Jaringan Quantus. Ia dirancang untuk memastikan keberlanjutan jangka panjang dan distribusi yang adil.

## Statistik Utama

- **Total Pasokan**: 1.000.000.000 QUAN.
- **Kegunaan**: Digunakan untuk biaya gas, staking, dan tata kelola.
- **Mekanisme Deflasi**: Sebagian dari setiap biaya transaksi dibakar.

## Distribusi

Kami telah mengalokasikan 50% dari total pasokan untuk hadiah penambangan selama 20 tahun ke depan untuk memastikan bahwa jaringan tetap terdesentralisasi untuk generasi mendatang.
