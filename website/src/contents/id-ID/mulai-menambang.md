# Mulai Menambang di Quantus

Quantus menggunakan mekanisme konsensus hibrida yang memungkinkan siapa pun dengan komputer standar untuk berpartisipasi dalam mengamankan jaringan.

## Persyaratan

- **CPU/GPU**: Sebagian besar perangkat keras modern mampu.
- **Node Quantus**: Unduh perangkat lunak terbaru dari GitHub kami.
- **Stabilitas**: Koneksi internet yang konsisten.

## Mengapa Menambang?

Dengan menambang, Anda membantu mendesentralisasikan jaringan dan melindunginya dari serangan. Sebagai imbalannya, Anda mendapatkan hadiah blok dalam bentuk token QUAN.

Bergabunglah dengan revolusi dan mulai amankan masa depan kuantum hari ini.
