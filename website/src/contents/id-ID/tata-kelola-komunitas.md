# Tata Kelola Komunitas

Quantus tidak dimiliki oleh korporasi; ia dimiliki oleh komunitas. Model tata kelola kami memastikan bahwa setiap pemegang QUAN memiliki suara.

## Cara Kerjanya

1. **Proposal**: Setiap anggota komunitas dapat mengajukan proposal.
2. **Diskusi**: Proposal diperdebatkan di forum terbuka kami.
3. **Voting**: Pemegang menggunakan token QUAN mereka untuk memberikan suara pada keputusan penting, mulai dari pemutakhiran protokol hingga alokasi treasury.

Kami percaya bahwa desentralisasi hanya mungkin terjadi ketika komunitas berada di kursi pengemudi.
