# Tokenomics de QUAN

El token QUAN es el alma de la red Quantus. Fue diseñado para garantizar la sostenibilidad a largo plazo y una distribución justa.

## Estadísticas Clave

- **Suministro Total**: 1.000.000.000 QUAN.
- **Utilidad**: Se utiliza para tarifas de gas, staking y gobernanza.
- **Mecanismo Deflacionario**: Una parte de cada tarifa de transacción se quema.

## Distribución

Hemos asignado el 50% del suministro total para recompensas de minería durante los próximos 20 años para garantizar que la red permanezca descentralizada para las generaciones venideras.
