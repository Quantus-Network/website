# Transacciones Reversibles

Uno de los mayores obstáculos para la adopción de las criptomonedas es el miedo a perder fondos debido a un solo error. Quantus resuelve esto con **Transacciones Reversibles**.

## Cómo Funciona

Cuando envías una transacción, puedes definir una "ventana de reversibilidad" (por ejemplo, 24 horas). Durante este tiempo, si te das cuenta de que cometiste un error o caíste en una estafa, puedes iniciar una reversión.

### Casos de Uso:
1. **Direcciones Mal Escritas**: Recupera tus fondos al instante.
2. **Disputas con Comerciantes**: Una función integrada similar a un depósito en garantía (escrow).
3. **Protección contra Estafas**: Intercepta transferencias fraudulentas antes de que se finalicen.

Esto es autocustodia con una red de seguridad.
