# 如何保护您的助记词

助记词是通往您整个数字财富的钥匙。在去中心化的世界里，没有“忘记密码”按钮。

## 最佳实践

1. **切勿数字化**：切勿将助记词存储在电脑、手机或云存储中。
2. **物理备份**：将其写在纸上或刻在金属上。
3. **分开放置**：将助记词的不同部分存储在独立的、安全的位置。
4. **守护者**：使用 Quantus 的“守护者”功能添加一层社交恢复。

记住：不是你的私钥，就不是你的资产。
