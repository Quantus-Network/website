# Umkehrbare Transaktionen

Eines der größten Hindernisse für die Krypto-Adoption ist die Angst, Gelder durch einen einzigen Fehler zu verlieren. Quantus löst dies mit **Umkehrbaren Transaktionen**.

## Wie es funktioniert

Wenn Sie eine Transaktion senden, können Sie ein „Umkehrbarkeitsfenster“ definieren (z. B. 24 Stunden). Wenn Sie in dieser Zeit feststellen, dass Sie einen Fehler gemacht haben oder auf einen Betrug hereingefallen sind, können Sie eine Umkehrung einleiten.

### Anwendungsfälle:
1. **Falsch getippte Adressen**: Erhalten Sie Ihr Geld sofort zurück.
2. **Händlerstreitigkeiten**: Eine integrierte Treuhandfunktion.
3. **Betrugsschutz**: Abfangen betrügerischer Überweisungen, bevor sie finalisiert werden.

Dies ist Selbstverwahrung mit Sicherheitsnetz.
