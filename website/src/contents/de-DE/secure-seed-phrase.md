# So sichern Sie Ihre Seed-Phrase

Ihre Seed-Phrase ist der Schlüssel zu Ihrem gesamten digitalen Reichtum. In einer dezentralen Welt gibt es keine „Passwort vergessen“-Schaltfläche.

## Best Practices

1. **Niemals digital**: Speichern Sie Ihre Seed-Phrase niemals auf einem Computer, Telefon oder im Cloud-Speicher.
2. **Physisches Backup**: Schreiben Sie sie auf Papier oder prägen Sie sie in Metall.
3. **Aufteilen**: Bewahren Sie verschiedene Teile der Phrase an separaten, sicheren Orten auf.
4. **Guardians**: Nutzen Sie die Quantus „Guardians“-Funktion, um eine Ebene der sozialen Wiederherstellung hinzuzufügen.

Denken Sie daran: Nicht Ihre Schlüssel, nicht Ihre Coins.
