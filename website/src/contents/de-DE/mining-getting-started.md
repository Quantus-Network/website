# Mining auf Quantus

Quantus verwendet einen hybriden Konsensmechanismus, der es jedem mit einem Standardcomputer ermöglicht, an der Sicherung des Netzwerks teilzunehmen.

## Anforderungen

- **CPU/GPU**: Die meiste moderne Hardware ist fähig.
- **Quantus Node**: Laden Sie die neueste Software von unserem GitHub herunter.
- **Stabilität**: Eine beständige Internetverbindung.

## Warum minen?

Durch das Mining helfen Sie, das Netzwerk zu dezentralisieren und es vor Angriffen zu schützen. Im Gegenzug verdienen Sie Blockbelohnungen in Form von QUAN-Token.

Schließen Sie sich der Revolution an und beginnen Sie noch heute, die Quantenzukunft zu sichern.
