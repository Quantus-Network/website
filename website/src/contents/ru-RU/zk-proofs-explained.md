# Объяснение ZK-Proofs

Доказательства с нулевым разглашением (ZK-Proofs) — это прорыв в криптографии, который позволяет одной стороне доказать другой, что утверждение истинно, не раскрывая никакой информации, кроме самой валидности утверждения.

## Как Quantus использует ZK-Proofs

В сети Quantus мы используем рекурсивные ZK-SNARK для сжатия всей истории блокчейна в крошечное доказательство.

### Преимущества:
- **Масштабируемость**: Узлам не нужно повторно запускать каждую транзакцию.
- **Конфиденциальность**: Транзакции могут быть проверены без раскрытия конфиденциальных данных.
- **Легкие узлы**: Запускайте узел на своем смартфоне!
