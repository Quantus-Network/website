# Обзор экосистемы: QuantumSwap

Мы рады сообщить, что **QuantumSwap** успешно запустил свою бета-версию в тестовой сети Quantus.

## Что такое QuantumSwap?

QuantumSwap — это автоматизированный маркет-мейкер (AMM), который использует наши квантово-безопасные подписи, чтобы гарантировать защиту поставщиков ликвидности от будущих квантовых угроз.

### Особенности:
- **Низкое проскальзывание**: Оптимизированные кривые связывания.
- **Пулы, защищенные PQC**: Безопасность для долгосрочных держателей.
- **Интегрированный кошелек**: Удобная работа с приложением Quantus.
