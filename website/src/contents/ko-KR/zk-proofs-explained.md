# ZK-Proofs 설명

제로 지식 증명(Zero-Knowledge Proofs, ZK-Proofs)은 한 당사자가 다른 당사자에게 진술의 유효성 외에는 어떤 정보도 공개하지 않고 해당 진술이 사실임을 증명할 수 있게 해주는 암호학의 획기적인 기술입니다.

## Quantus의 ZK-Proofs 활용 방법

Quantus 네트워크에서는 재귀적 ZK-SNARK를 사용하여 블록체인의 전체 기록을 아주 작은 증명으로 압축합니다.

### 장점:
- **엄청난 확장성**: 노드가 모든 거래를 다시 실행할 필요가 없습니다.
- **프라이버시**: 민감한 데이터를 노출하지 않고 거래를 검증할 수 있습니다.
- **경량 노드**: 스마트폰에서 노드를 실행하세요!

재귀적 SNARK 아키텍처에 대한 기술적 심층 분석을 기대해 주세요.
