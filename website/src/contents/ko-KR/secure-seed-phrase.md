# 시드 구문을 안전하게 보관하는 방법

시드 구문은 귀하의 모든 디지털 자산의 열쇠입니다. 탈중앙화된 세상에서 "비밀번호 찾기" 버튼은 없습니다.

## 모범 사례

1. **디지털화 금지**: 시드 구문을 컴퓨터, 휴대폰 또는 클라우드 저장소에 절대 저장하지 마세요.
2. **물리적 백업**: 종이에 적거나 금속에 새기세요.
3. **분할 보관**: 구문의 각 부분을 별도의 안전한 장소에 나누어 보관하세요.
4. **가디언**: Quantus의 "가디언" 기능을 사용하여 소셜 복구 계층을 추가하세요.

기억하세요: 키가 없으면 당신의 코인도 아닙니다.
